# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Books

- `GET /api/books`
- `GET /api/books/:id`

### Bookshelves

- `GET /api/bookshelves`
- `POST /api/bookshelves`
- `GET /api/bookshelves/:id`
- `PATCH /api/bookshelves/:id`
- `DELETE /api/bookshelves/:id`
